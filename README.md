
# Xen

a minimalistic unblocker focused on usability and simplicity                                         .                                        

a nova labs project.




## Deployment

To deploy this project run

```bash
  git clone https://github.com/xenongh/Xen.git
  cd Xen
  npm i
  node index.mjs
```

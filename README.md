
# Xen

a minimalistic unblocker focused on usability and simplicity                                         .                                        

a nova labs project.




## Features

- Games Library
- Plugin Library
- Easy to Use Unblocker
- Customization
- Tab Cloaking
- About:Blank Cloaking
- Inspect Element
- JS Injection
- Fast Speeds
- Ad-Free

## Deployment

To deploy this project run

```bash
  git clone https://github.com/xenongh/Xen.git
  cd Xen
  npm i
  node index.mjs
```
